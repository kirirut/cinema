# Cinema

Онлайн-кинотеатр: каталог фильмов, регистрация пользователей, оценки, отзывы и избранное.

## Стек

- Java 17, Maven
- PostgreSQL
- **React** — `frontend/` (ветка `react`)
- **Angular 19** — `angular-app/` (ветка `angular`)

### Angular-фронтенд

```bash
cd angular-app
npm install
npm start
```

→ http://localhost:4200 (прокси API на `:8080`)
